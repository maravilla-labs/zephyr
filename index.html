API Usage

// Clear all cache
await zephyr.clear();

// Clear by pattern
await zephyr.clearPattern('.*\\.jpg$');

// Get statistics
const stats = await zephyr.stats();
console.log(stats);

// Toggle debug mode
await zephyr.debug();
Tip: Add ?zephyrDebug=true to the URL to enable debug logging in the console.
